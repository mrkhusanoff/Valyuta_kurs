
## ⚙️ .env fayl
Loyihangiz papkasiga `.env` fayl yarating va quyidagini yozing:

```
BOT_TOKEN=YOUR_BOT_TOKEN_HERE
```

Railway yoki Render’da esa bu qiymatni “Environment Variables” bo‘limida `BOT_TOKEN` nomi bilan qo‘shing.

## 🚀 Ishga tushirish
```
python main.py
```
